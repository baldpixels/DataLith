Handaxe

The Handaxe scours a website for dead links, chopping them off as it goes. (i.e. It returns the ratio of dead links to active links.)

Hook

The Hook fishes for links, realing them in as fast as it can. (i.e. It returns the mean time required to load a website's links.)

Grindstone

The Grindstone pounds a website's images into a flat mold. (i.e. It returns a collage of the site's images.)
